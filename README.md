# Random Username Generator (rug)

## Installation

```
go install github.com/WangYihang/rug@latest
```

## Usage

```
$ rug -h
Usage:
  rug [OPTIONS]

Application Options:
  -c, --count=   Number of usernames to generate (default: 1)
  -v, --version  Show version

Help Options:
  -h, --help     Show this help message
```

### Generate a single username

```
$ rug
VittaLibra66
```
